# todo-list-test

## Demo 
https://floating-ridge-60618.herokuapp.com

## Technologies
- React
- React hooks
- React context-API
- JSX
- NodeJS
- ExpressJS
- Mongoose

## Features
- Admin can edit/detete/add movie data
- User can see list of movies and filter data
- For admin rights user can login by entering role 'admin'
